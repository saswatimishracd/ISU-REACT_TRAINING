CBOI QR Code Generation API
Endpoint Description
This endpoint is used to convert a QR code into a Base64 encoded string. It is particularly useful for applications that require QR code data to be transmitted in a compact format.
API Details
Method: POST
URL: https://api-preprod.txninfra.com/encrV4/CBOI/merchant/qr_convert_to_base64
Request Body: The request must contain a JSON object with the following parameter:
RequestData (string): This parameter should include the QR code data that needs to be converted into Base64 format.
Request Headers: 
pass_key: Should contain the pass_key used for CBOI UAT
Authorization: Should contain the Authentik token used for CBOI UAT

Response Description
Status Code: 200 OK
Content-Type: application/json
Response Body: The response will return a JSON object containing:
ResponseData (string): This field will contain the Base64 encoded string of the QR code if the conversion is successful. In this case, it is currently returned as an empty string.
Notes
Ensure that the RequestData is properly formatted as a valid QR code string for successful conversion.
The endpoint is designed to handle large payloads, but it is recommended to keep the data within reasonable limits to avoid performance issues.
