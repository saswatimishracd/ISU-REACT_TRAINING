CBOI - Merchant Onboarding API

URL
Staging: 
UAT : https://services-cboi-uat.isupay.in/CBOI/merchant/bulk-onboard
PROD: https://centralbank-prod.isupay.in/CBOI/merchant/bulk-onboard

Configurations
Method: POST
Headers: Authorization token
Description: API for onboarding  merchants of CBOI
Request Body:

Original
{ 
 "entityId": "centralbank",
 "mobileNo": "9292123456", 
 "paymentAddress": "9292123456@cbin", 
 "merchantAccountNo": "9292123456", 
 "accountType": "CURRENT", 
 "ifsc": "CBINO908392", 
 "merchantLegalName": "Test User 4", 
 "channelId": "UPI", 
 "mcc": "9399", 
 "gstIn": "27ABCDE1234F1Z8", 
 "storeId": "123459", 
 "merchantGenre": "OFFLINE", 
 "onboardingType": "BANK", 
 "isVerified": "N", 
 "address": "Patia, Bhubaneswar", 
 "city": "Bhubaneswar", 
 "state": "Odisha", 
 "pincode": 751031 
 }

Sample Response
{
    "status": 0,
    "message": "Merchant onboarded successfully"
}



NOTE

Every field is mandatory except gstIn
